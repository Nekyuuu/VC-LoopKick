<h1>Compilation</h1>

To compile a dotnet project, using dotnet.

```bash
dotnet publish -r win-x64 --self-contained=false /p:PublishSingleFile=true
```
<br>
To compile the Golang DLL.

```bash
go build -o vc.dll -buildmode=c-shared dll.go
```
